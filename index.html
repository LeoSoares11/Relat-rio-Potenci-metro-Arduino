1. INTRODUÇÃO

Este relatório descreve o projeto de um LED controlado por um potenciômetro utilizando Arduino, com o objetivo de demonstrar o controle de intensidade luminosa.

2. OBJETIVOS

  • Controlar a intensidade luminosa de um LED usando um potenciômetro.
  • Compreender a leitura de valores analógicos com Arduino.

3. MATERIAIS UTILIZADOS

  • Arduino Uno
  • 1 LED
  • 1 Potenciômetro (10kΩ)
  • 1 Resistor (220Ω)
  • Protoboard
  • Fios de conexão

4. DIAGRAMA DO CIRCUITO

Abaixo está o diagrama do circuito projetado no Tinkercad:

5. CÓDIGO FONTE

int potenciometro = A0;
int led = 3;
int valorLido = 0;

void setup() {
    pinMode(potenciometro, INPUT);
    pinMode(led, OUTPUT);
    Serial.begin(9600);
}

void loop() {
    valorLido = map(analogRead(potenciometro), 0, 1023, 0, 255);
    analogWrite(led, valorLido);
    Serial.println(valorLido);
    delay(10);
}

6. PROCEDIMENTOS

O LED foi conectado ao pino digital do Arduino, e o potenciômetro ao pino analógico. O código foi carregado no Arduino e testado.

7. RESULTADOS

O LED variou sua intensidade luminosa de acordo com a posição do potenciômetro, demonstrando o controle efetivo.

8. DISCUSSÃO

O projeto funcionou conforme esperado, permitindo uma boa interação com o hardware e compreensão do controle analógico.

9. CONCLUSÃO

O projeto foi bem-sucedido, permitindo a prática de conceitos de eletrônica e programação com Arduino.
